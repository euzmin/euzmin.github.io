题意：求有多少个n*n矩阵满足下列条件

  • Ai, j ∈ {0, 1, 2} for all 1 ≤ i, j ≤ n.
  • Ai, j = Aj, i for all 1 ≤ i, j ≤ n.
  • Ai, 1 + Ai, 2 + … + Ai, n = 2 for all 1 ≤ i ≤ n.
  • A1, 1 = A2, 2 = … = An, n = 0.

解题思路：根据上面的条件我们会发现这个矩阵其实可以类比图的邻接矩阵。每个点的度为2相当于该点在一个环内。

题意：长度为n的字符串，删掉m个字符后有多少种不同的字串。

解题思路：dp[i][j]表示加入第i个数字后，总共删掉j个数字时，有多少种不同的序列。假设不考虑有重复的情况，dp方程为：dp[i][j]=dp[i-1][j] (第i个数字不删)+dp[i-1][j-1] (第i个数字删)。
